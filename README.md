## Эмбеддинги валлийского языка
[Ссылка на гугл-папку](https://drive.google.com/drive/folders/12tuKKCQ-CtlGuSaTlkP-G6CbIoxacKtl?usp=drive_link)
[Ссылка на оригинальный корпус](https://research.cardiff.ac.uk/converis/portal/detail/Dataset/119878310?auxfun=&lang=en_GB)

В папке:
- `embeddings.zip` - архив с cbow и skipgram эмбеддингами (размерность 100)
- `complete_corpus.zip` - архив с оригинальными текстами (не все из них используются)
- `lemmatized_corpus.txt` - итоговый лемматизированный корпус, на котором строились эмбеддинги
- `titles_list.txt` - список названий текстов, которые вошли в итоговый корпус
- `written.tsv` - оригинальные названия литературных текстов

В репозитории:
- `corpora.ipynb` - код со сбором корпуса
